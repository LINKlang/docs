## Clewd 简介

Clewd 是一款海外作者写的NodeJs应用，作用是将OpenAI格式请求转为官网请求用于和 [Claude](https://claude.ai/) 进行对话，并将收到的请求转回OpenAI格式用于和酒馆通讯
::: note 用某个名人的一句话就是

酒馆是使用AI的场所，小克(Claude)是AI，Clewd是拐骗良家小克到酒馆的黄毛，破限是酒

:::
